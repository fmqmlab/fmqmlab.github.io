Welcome to PyDPE-Web!#

This website hosts the web-based version of the PyDPE package developed by the Fahad Mahmood Group. This package plots a raw image file from our ARTOF detector, and shows the regions of the Brillouin zone where double photon emission might be observed. PyDPE-Web has no dependencies and can be run directly on your browser. Click on “QuickStart” to try it out!

If you would prefer to download the original PyDPE package locally, you can do so by heading to the “Additional Resources” section of our Documentation.

Warning

PyDPE-Web supports most major browsers and browser versions. However, there are open issues with specific versions of Safari (v14) that we are working on fixing. If you have the latest version of Safari, you should see no issues. For the best experience, please consider switching your browser to Google Chrome (or any other chromium based browser)
